Manajemen Data Siswa

Panduan Import Data Siswa

  1. Klik Download Template untuk mengunduh file template_siswa.csv.
  2. Buka file tersebut (di Excel/Google Sheets), isi data siswa Anda, lalu Simpan kembali sebagai .csv.
  3. Klik Import Data Siswa, pilih file .csv yang sudah Anda isi.

Penting: Jangan mengubah urutan kolom di template dan pastikan format tanggal adalah YYYY-MM-DD (contoh: 2010-05-17).
